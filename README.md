#  Ocean Tap

Ocean Tap adalah game arkade sederhana berbasis Python yang terinspirasi dari Flappy Bird, namun dengan tema bawah laut. Pemain mengendalikan sebuah kapal selam untuk melewati rintangan pipa-pipa di dalam laut tanpa menabrak.

---

##  Fitur Game
* **Tema Bawah Laut:** Menggunakan aset kapal selam dan rintangan yang disesuaikan.
* **Sistem Menu Interaktif:** Dilengkapi dengan Main Menu untuk memulai atau keluar dari game menggunakan keyboard.
* **High Score System:** Skor tertinggi otomatis tersimpan di dalam file `highscore.txt` sehingga rekor tidak hilang saat game ditutup.
* **Audio & Musik:** Dilengkapi dengan latar musik (BGM) yang seru dan efek suara saat melompat.
* **Efek Visual Shadow Text:** Tampilan teks UI yang lebih ciamik dengan efek bayangan.

---

##  Kontrol Game
* **Di Main Menu:** Gunakan `Panah Atas / Bawah` untuk memilih menu, dan tombol `Enter` atau `Space` untuk memilih.
* **Saat Bermain:** Tekan `Spacebar`, `Tombol X`, atau `Panah Atas` untuk membuat kapal selam bergerak naik.
* **Di Layar Game Over:** Tekan `Spacebar` untuk langsung main lagi, atau tekan tombol `M` untuk kembali ke Menu Utama.

---

##  Cara Install & Menjalankan

```bash
# 1. Clone repositori dan masuk ke foldernya
git clone https://github.com/USERNAME_KAMU/NAMA_REPOSITORI_KAMU.git
cd NAMA_REPOSITORI_KAMU


# 2. Install semua library yang dibutuhkan
pip install -r requirements.txt

# 3. Jalankan gamenya
python Ocean_tap.py
'''
